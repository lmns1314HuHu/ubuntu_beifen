Preload

从软件名字上看都知道这是一款通过预加载方式来提高性能的工具。

Preload监控应用在运行后会对数据进行分析，进而预测用户可能运行的应用程序并提前将二进制数据及相关信息载入内存，最终实现启动速度提升。

安装Preload能显著改善Ubuntu Unity全局性能。从某种程度上说，像OS Luna基础版这样的发行版也是依靠Preload才实现如此顺畅的操作感受,但是安装Preload可能会令操作系统的启动时间略微延长，有得必有失嘛。
安装：sudo apt-get install preload

用过Ubuntu的童鞋都知道，Ubuntu在独显方面的支持确实不尽人意。刚装好系统的时候，笔记本风扇不停的吹，持续发热，续航能力严重下降，连Linus都对Nvidia竖起中指。开源的驱动确实不是很完美。我之前一直用的使 bumblebee,选择关闭独显来勉强解决。有兴趣可以看看：http://bumblebee-project.org/index.html。

安装方法：

不过，有一个prime-indicator的软件，可以不用输入命令而在ＧＰＵ指奸切换功能，并且可以显示当前使用的GPU状态。可以通过PPA来安装。

如果安装之后发现无法切换的话，可能使因为没有禁止开源显卡驱动Nouveau。可以通过管理员权限修改/etc/modprobe.d/blacklist，如果 /etc/modprobe.d/ 下有 nouveau-blacklist.conf，则编辑这个文件。在文件末尾添加 blacklist nouveau 。

修改Swappiness值

在ubuntu 里面,swappiness的值的大小对如何使用swap分区是有着很大的联系的.swappiness=0的时候表示最大限度使用物理内存,然后才是 swap空间,swappiness＝100的时候表示积极的使用swap分区,并且把内存上的数据及时的搬运到swap空间里面.两个极端,对于 ubuntu的默认设置,这个值等于60,建议修改为10.具体这样做：

sudo vim /etc/sysctl.conf
在最后一行加上vm.swappiness=10

BCloud—百度网盘Linux版

bcloud - 百度网盘的 Gtk/Linux 客户端，由LiuLang(供职deepin linux)开发，通过百度PCS接口实现，基本可以完整网盘各项操作。DropBox虽然好，但是在比较和谐的网络环境下还是使用国内的方便。

1.支持流媒体模式, 可以直接用本地的视频播放软件播放百度网盘里的视频. 百度网盘的网页版及windows客户端都是使用的流媒体模式来播放的网盘视频, 它们会调用adobe flash player, 这东西很占CPU, 不一会儿笔记本就很热了; 但bcloud调用本地的视频播放器, 系统消耗就很少了. 朋友们可以比较一下性能.
2.与本地程序有很好的整合, 可以直接双击打开网盘里的文件(使用本地程序).
2.下载整个目录, 也可以把本地文件夹拖到bcloud里面来上传整个目录; 但还不支同步.
4.整合了离线下载.

如果结合IPv6，会达到10Mbs/s的速度，这里使百度PCS的IPV6地址,添加到/etc/hosts/中

影音集结号MvGather

影视集结号(MvGather)是专为ubuntu及衍生版用户打造的播放器，内容来源于云帆搜索，版权归原网站所有。
1,它是个集合在线视频的工具，内容来自各大视频网站，可在线播放优酷、土豆、奇艺、风行等国内各网站视频，而且不再有广告。只要将要播放的页面地址粘贴，就可以自动找到视频。
2,它可以当做播放本地视频的简易播放器。
3,在网友推荐部分，汇聚了当前最热的大片。
4,磁力搜索部分，神马都能搜，配合百度云盘，无敌了。请合法使用。

Haroopad — Markdown编辑器

Haroopad是一款Linux上的markdown编辑器，使用Chromium作为UI，支持Windows、Mac OS X和Linux。主题样式丰富，语法标亮支持54种编程语言。 如下图所示，一边是代码编辑窗口，一边是预览窗口，可以实时更新。其邮件导出功能可以将文档发送到Tumblr和Evernote。
在写这篇文档的时候我就是通过它来预览的，并且支持VIM编辑模式。

下载地址：https://bitbucket.org/rhiokim/haroopad-download/downloads/haroopad-v0.13.1-x64.deb

VMWare

在Ubuntu下我安装了两种虚拟机软件，Ubuntu默认使用的VBOX，还有就是VMWare。由于经常做一些测试，不方便折腾自己的桌面环境，所以索性直接再在虚拟机里面安装一个Linux环境或者Windows环境。在实验过程中发现，VMware由于vmware tools对虚拟机中系统的支持较好，支持全屏、虚拟机和宿主机文件拖拉贡献、独占模式、双屏支持等方面表现较好。
而且在VMWare中安装windwos的执行效率也很快（++我的电脑配置：2.5G主频、双核4线程、8G内存++）。相比windows中使用VMWare安装Linux，效率更高。
尤其对于windows系统，可以使用Unity，给你在linux桌面环境中使用windows程序的感觉。

下载地址：https://my.vmware.com/web/vmware/details?downloadGroup=WKST-1110-LX&productId=462&rPId=7457
